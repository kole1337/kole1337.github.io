Ring Election Algorithm Implementation
The aim of this coursework is to develop an algorithm that will manage leader election for a Ring System, where processes can join. This implementation follows standard TCP protocols in utilising connection between processes, joining a "ring circle", and starting an election, with the associated processes of considering a victor and communication between neighbours once a leader is decided.

Clients communicate via TCP port connection. The clients must be connected on the same network.
